💡 如何使用生成的 ZIP 包？

1. 點擊「下載遊戲 ZIP 包」按鈕，將會下載一個壓縮檔。

2. 解壓縮此檔案，您會得到一個 `index.html`、一個 `item/` 資料夾。

3. 將這些檔案和資料夾保持相對路徑放置。

4. 使用任何現代瀏覽器打開 `index.html` 即可開始遊戲！

重要提示：
如果您在 `file://` 協定（直接從硬碟打開 HTML 檔案）下運行此生成系統，
預設物品的圖片預覽可能因瀏覽器安全限制而無法顯示。
為確保預覽正常，請將 `generator.html` 放置在一個具有 `item/` 資料夾（內含預設 SVG 圖片）的根目錄下，或者使用本地網路伺服器運行本頁面。例如：
  - 安裝 VS Code 的 "Live Server" 擴充功能，
      然後對 `generator.html` "Open with Live Server"。
  - 在終端機中，切換到此檔案目錄，執行 `python -m http.server`，
      然後瀏覽 `http://localhost:8000/generator.html`。
